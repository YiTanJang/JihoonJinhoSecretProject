# SAmultithread Runtime Instructions

1.  **Unzip** this package on your computer.
2.  **Run the Solver:**
    *   Go to the `bin` folder.
    *   Double-click `SAmultithread_4d.exe` OR run it from a terminal (PowerShell/CMD).
    *   It will automatically create a database in the `db` folder and start working.
3.  **Requirements:**
    *   If you get a "Missing DLL" error (like VCRUNTIME140.dll), please install the **Visual C++ Redistributable (x64)** from Microsoft.
4.  **Transferring Results:**
    *   Stop the program.
    *   Copy `db/optimizer_4d.db` back to your main computer.
    *   Use the merge script on your main computer to import the results.
